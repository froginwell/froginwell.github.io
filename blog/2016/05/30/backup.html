常用快捷键备忘

发表于 2016-05-30 | 分类于 快捷键

包括chrome、vimium、shell、tmux的常用按键

1. chrome常用快捷键

快捷键 作用
Ctrl+n 打开新窗口
Ctrl+t 打开新标签页
Ctrl+w 或 Ctrl+F4 关闭当前标签页或弹出式窗口
Ctrl+Shift+t 重新打开上次关闭的标签页。谷歌浏览器可记住关闭的最后10个标签页
Ctrl+1-8 切换到指定标签页
Ctrl+9 切换到最后一个标签页
Ctrl+Tab 切换到下一个标签页
Ctrl+Shift+Tab 切换到上一个标签页
按住Ctrl点击链接 在新标签页中打开链接
按住Shift点击链接 在新窗口中打开链接
Ctrl+l或F6 突出显示网址区域中的内容
输入网址后按Alt+Enter 在新标签页中打开网址
Ctrl+h 查看历史记录页
Ctrl+j 查看下载页
Ctrl+d 将当前网页加入书签
Ctrl++ 放大文字
Ctrl– 缩小文字
Ctrl+o 还原到正常文字大小

2. vimium常用按键

按键 作用
j 向下滚动一行
k 向上滚动一行
d 向下滚动半屏
u 向上滚动半屏
gg 回到顶部
G 到达页面底部
J 下一个标签页
K 上一个标签页
H 后退
L 前进
f 将当前网页上的所有可见链接/输入框分配一个快捷键
g+i 将光标定位到第一个输入框，可用Tab切换
x 关闭当前页面
X 恢复刚刚关闭的页面
o 相当于chrome的地址栏，可以匹配历史记录、收藏夹并在当前窗口打开
O 在新窗口打开，其他同o
r 重新载入当前网页
shift+/ 快捷键大全

3. shell常用组合键

按键 作用
Ctrl+w 回删一个单词
Ctrl+u 删除当前光标到行首的部分
Ctrl+k 删除当前光标到行尾的部分
Alt+b 按单词向行首移到光标
Alt+f 按单词向行尾移动光标
Ctrl+a 移到光标到行首
Ctrl+e 移到光标到行尾
Alt+. 循环显示输过的命令的参数
Ctrl+x Ctrl+e 打开设置的编辑器，ubuntu中可在.bashrc中加入 export EDITOR=vim 来配置vim为默认的编辑器
history 查看最近执行的命令
!n 执行历史记录中的第n条命令
!$ 上一条命令的最有一个参数
!! 执行最后一条命令
sudo !! 以root用户执行最后一条命令
pstree -p 以树形查看进程

4. tmux常用命令

未进入tmux时的命令

命令 作用
tmux new -s session-name 新建一个名字为session-name的会话
tmux a -t session-name 接入指定会话
tmux kill-session -t session-name 结束会话

进入tmux时的命令

会话相关命令，先按Ctrl+b，然后按下表中的按键

按键 作用
按:，然后输入new -s session-name按 新建会话
s 显示所有会话，按会话前的数字标号可以进入相应会话
$ 重命名当前会话
d 从会话断开

窗口相关命令，先按Ctrl+b，然后按下表中的按键

按键 作用
c 创建新窗口
, 重命名当前窗口
w 列出所有窗口
% 竖直分割窗口
“ 水平分割窗口
n 下一个窗口
p 上一个窗口
0-9 0-9号窗口

窗格相关命令，先按Ctrl+b，然后按下表中的按键

按键 作用
% 竖直分割窗格
“ 水平分割窗格
h 切到左侧窗格
j 切到下侧窗格
k 切到上侧窗格
l 切到左侧窗格
q 显示作用窗格
o 在窗格间轮流切换
} 与下一个窗格交换位置
{ 与上一个窗格交换位置
! 在新窗口中显示当前窗格
x 关闭当前窗格

复制粘贴相关

  • Ctrl+b [ 进入复制模式
  • 空格 开始选区
  • 然后可以用h、j、k、l等选择选区
  • Enter 确定选择
  • Ctrl+b ] 粘贴
