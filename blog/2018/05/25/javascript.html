3. Javascript new关键字

用new来调用函数，会自动执行下面的过程：

  1. 创建一个全新的对象
  2. 这个新对象会被执行原型连接
  3. 这个新对象会绑定到函数调用的this
  4. 如果函数没有返回其他对象，那么new表达式中的函数调用会自动返回这个新对象，否则返回return指明的对象
